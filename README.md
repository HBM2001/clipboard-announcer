# Clipboard Announcer

Clipboard Announcer is an NVDA add-on that provides spoken feedback for common editing and clipboard actions.

## Details

- Author: H2k1
- Version: 1.2
- Minimum NVDA version: 2024.1
- Last tested NVDA version: 2026.1

## Project Structure

- `manifest.ini`: Add-on metadata
- `globalPlugins/`: NVDA global plugin source
- `locale/`: Localization files
